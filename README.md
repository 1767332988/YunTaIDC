## 安装使用
### 环境要求：
- PHP版本 = 7.4【官方测试版本，其他版本不保证】
- .htaccess支持
### 安装步骤：
1. 上传源码到根目录并绑定域名
2. 浏览域名/install，例如：`http://www.example.com/install/`
3. 填写相关数据库信息点击完成安装
4. 安装完成
### 更新系统：
1. 下载更新包并且上传至网站根目录
2. 访问域名/install/update.php，例如：`http://www.example.com/install/update.php`
3. 更新完成
